Revertigo

An ongiong personal project built off of the 3D tower defense maze building proof-of-concept!

The pathfinding algorithm uses a multitude of hash tables, caching, and square-root work-arounds to vastly decrease runtime and allow for the player to actively place blocks to redirect the path

Everytime you redirect the path it will search through all the path nodes for chokepoints and pre-process positions where placing a block would block the path entirely
